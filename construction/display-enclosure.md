# Display Enclosure

Waterproof, tamper-resistant PVC enclosure for the 64×32 RGB LED panel (Matrix Portal ESP32-S3).

## Dimensions

- Interior: 15.2" x 7.5" x 2.5" (panel + Matrix Portal + power bank)
- Frame wall thickness: 3/4" (1x4 boards)
- Rear panel thickness: 1/2"
- Exterior: 17" x 9" x 3.5"

## Components

### Structure
- PVC 1x4 boards (top, bottom, two sides)
- PVC 1/2" sheet (rear panel, cut to 17" x 9")
- Polycarbonate sheet (front window, already have)

### Joints
- PVC cement — top, bottom, and permanent side to rear panel
- Pin-in-hex security screws + brass inserts — access side panel only, resealed with silicone each time

### Mounting
- Eyebolts through rear panel + zip ties around fence rails (stabilization)
- Long shackle padlock through hasp staple, looped around fence rail

### Sealing
- Silicone caulk — all seams, corners, front window perimeter, access panel
- Silica gel desiccant inside

### Tamper Resistance
- Hasp on rear panel, long shackle padlock through hasp staple and around fence rail — prevents removal
- Pin-in-hex security screws on access side panel — prevents opening
- Hasp and eyebolts must go through rear panel into a backing plate — 1/2" PVC alone won't hold pull force

## Assembly Notes

- Pre-drill all holes before screwing
- Use stainless or coated screws for anything exposed
- Leave ~1cm airspace between LED panel and box walls (heat)
- Countersink security screw holes so heads sit flush
